编程相关书籍的阅读推荐

2025-03-04 JavaScript►TypeScript►Node.js►Python►阅读 JavaScript, Node.js, Python, TypeScript, 阅读 评论

本篇博文主要是来推荐学习 JavaScript TypeScript Node.js Python 编程语言相关的书籍。

评价、口碑、ISBN等书籍相关信息查阅: 豆瓣读书

购买前，切记一定要先看评价！自己究竟适不适合阅读，然后再决定是否进行购买。

一、JavaScript

  1. 《JavaScript 高级程序设计 第4版》 ISBN: 9787115545381 出版年: 2020
  2. 《JavaScript 高级程序设计 第5版》 ISBN: 9787115655851 出版年: 2024
  3. 《JavaScript 权威指南 第7版》 ISBN: 9787111677222 出版年: 2021
  4. 《你不知道的 JavaScript 上》 ISBN: 9787115385734 出版年: 2015
  5. 《你不知道的 JavaScript 中》 ISBN: 9787115431165 出版年: 2016
  6. 《你不知道的 JavaScript 下》 ISBN: 9787115471659 出版年: 2018
  7. 《JavaScript DOM编程艺术 第2版》 ISBN: 9787115249999 出版年: 2011

二、TypeScript

  1. 《TypeScript编程》 ISBN: 9787519845964 出版年: 2020
  2. 《TypeScript 入门与实战》 ISBN: 9787111669722 出版年: 2020
  3. 《TypeScript 实战指南》 ISBN: 9787111626701 出版年: 2019
  4. 《深入理解TypeScript》 ISBN: 9787121376511 出版年: 2020

三、Node.js

  1. 《深入浅出 Node.js》 ISBN: 9787115335500 出版年: 2013
  2. 《Node.js 实战》 ISBN: 9787115352460 出版年: 2014

四、Python

  1. 《Python编程 第3版 从入门到实践》 ISBN: 9787115613639 出版年: 2023
  2. 《笨办法学 Python 3》 ISBN: 9787115478818 出版年: 2018
  3. 《“笨办法”学Python 第3版》 ISBN: 9787115350541 出版年: 2014
  4. 《Python Cookbook 第3版》 ISBN: 9787115379597 出版年: 2015
  5. 《流畅的Python》 ISBN: 9787115454157 出版年: 2017
  6. 《流畅的Python 第2版》 ISBN: 9787115612366 出版年: 2023
